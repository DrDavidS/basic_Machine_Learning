# 机器学习快速入门要点

本仓库的最初建立目的是给实验室的师弟师妹们讲课用的，目的是快速掌握机器学习实践的基本要领，注重从理论到实战的转换。

## 学前须知

### 本入门教程合适以下人群

- 熟悉线性代数
- 熟悉概率统计
- 有一定的编程基础，语言不限
- 刚刚学完机器学习/模式识别/统计学习方法，想要快速上手做研究的人

### 本入门教程不合适

- 对机器学习理论算法一无所知的人
- 机器学习熟练者

## 学习路线

1. Python基础
2. Numpy基础
3. Pandas基础
4. Python简单绘图技巧
5. 基本的机器学习框架

## 无法浏览怎么办？

无法直接在GitHub上加载ipynb格式的话，可以下载文件在本地使用Jupyter Notebook浏览，或者点击使用[nbviewer](https://nbviewer.jupyter.org/)浏览。
